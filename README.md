# CSCE313_Project3_4

# Back end 
Pre-requisite:
```
cd backend
npm install
```

Start server:
```
npm start
```

Dev locally on MacOS:
```
npm run dev-mac
```

Dev locally on Windows:
```
npm run dev-win
```

# Front end
Pre-requisite:
```
cd frontend
npm install
```

Start client:
```
npm start
```
